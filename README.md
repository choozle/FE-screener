# FE Screener

### Instructions:

- You'll be walking your interviewer through this exercise
  - Be specific with your instructions
  - Don't hesitate to ask questions
- Checkout a new branch matching the pattern `your-name-DD/MM/YYYY`
- Start the local development server.
- Run the tests.
- Make fixes where necessary.
- When finished, make a commit with a message of your choosing.
- Push changes to your branch.
- That's it!
